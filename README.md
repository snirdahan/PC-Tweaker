## 🚀 About

**PC Tweaker** is a lightweight tool designed to improve system performance and responsiveness.
It applies smart optimizations to help your PC run faster, smoother, and more efficiently.

Built for gamers and power users who want real performance gains without unnecessary bloat.

---

## ⚙️ Features

* ⚡ Performance optimization
* 🎯 Reduced input lag
* 🧹 Cleaner system processes
* 🚀 Faster response times
* 🔒 Safe and minimal tweaks
